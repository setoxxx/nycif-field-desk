# NYCIF Admin Dashboard (v0 skeleton)

Read-only static admin page for operator visibility. Published at `/admin/` on GitHub Pages when deployed from repo root.

## v0 scope

- Removes the planned `/admin/` GitHub Pages 404 after merge/deploy
- Static HTML only — no write controls, deploy buttons, auth, or secrets
- Placeholder panels only — no live event data loading yet
- Does not decide what appears on the public map
